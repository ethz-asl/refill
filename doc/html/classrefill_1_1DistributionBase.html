Detailed Description

template<typename DERIVED>
class refill::DistributionBase< DERIVED >

Class that implements the CRTP.

Class that implements the Curiously Recurring Templating Pattern so the clone function doesn't have to be implemented in every derived distribution.

For new distributions, inherit from this class like this:

class NewDistribution : public DistributionBase<NewDistribution>
