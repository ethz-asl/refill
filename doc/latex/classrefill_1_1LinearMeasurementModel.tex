\subsection{Detailed Description}
Class that implements a linear measurement model. 

This class is an implementation of linear measurement models of the form\+:

$ y_k = H_k \cdot x_k + M_k \cdot w_k $

Where $ y_k $ is the measurement, $ x_k $ the system state, $ w_k $ the measurement noise, $ H_k $ the measurement mapping and $ M_k $ the noise mapping at time step $ k $.

Use this class together with the \hyperlink{classrefill_1_1LinearSystemModel}{Linear\+System\+Model}, noise of type \hyperlink{classrefill_1_1GaussianDistribution}{Gaussian\+Distribution} and the \hyperlink{classrefill_1_1ExtendedKalmanFilter}{Extended\+Kalman\+Filter} if you want to implement a simple kalman filter.
